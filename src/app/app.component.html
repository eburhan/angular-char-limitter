Parametre Açıklaması Varsayılan Değer Değer Türü
clMaximum Girdi alanına yazılabilecek maksimum karakter sayısını belirler. zorunludur 255 number
clTemplate Kalan karakter mesajının ekranda nasıl görüntüleceğini belirten şablondur.
  • {{ '{0}' }} maksimum izin verilen karakter sayısı
  • {{ '{1}' }} girilmiş toplam karakter sayısı
  • {{ '{2}' }} geriye kalan karakter sayısı
 {{ '{0}/{1}' }} string
clElement Girdi alanının altına eklenecek olan html elementinin türünü belirtir. DIV veya SPAN gibi istediğiniz türde herhangi bir html elementi bu sayede belirtilebilir. div string
clClass "clElement" ile yaratılan html elementine class vermek için kullanılır. char-limitter string
clClassOver izin verilen karakter sayısı aşılmış ise "clElement" ile yaratılan html elementine bu class eklenir. varsayılan davranışta, izin verilen karakter sayısı aşıldığında Kalan Karakter yazısını kırmızı renkte görmektesiniz. char-limitter-over string
clClassFlush eğer Kalan Karakter mesajı yanıp sönsün isteniyorsa (bknz: clFlush) bu esnada kalan karekter mesajına uygulanacak olan class ismini belirtir. char-limitter-over-flush string
clClassInput girdi alanına (textarea veya input...) eklenen class ismidir. char-limitter-input string
clFlush girdi alanına girilebilecek karakter kalmadığını daha da vurgulamak amacıyla mesajı yakıp söndüren bir efekt uygulansın mı uygulanmasın mı? false booelan
